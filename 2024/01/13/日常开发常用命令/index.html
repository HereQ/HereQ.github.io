记录日常工作中使用到的linux命令，方便查找；日常使用时的场景是多种工具相互配合，所以就都写到一个文档中了~

1.文本三剑客

1.1 grep

命令 含义
-rn 在当前目录下递归查找(r)，并显示行号(n)
-w 精准匹配
-v 不包含
-i 不区分大小写

1.2 awk

echo “$txt” | grep “dt=” | awk -F’/‘ -v a=”$level” ‘{print $a}’ | awk -F’=’ ‘{print $2}’ | sort | tail -n 1

命令 含义
awk -F ‘:’ ‘{print $2}’ 对每行按照:进行分割，打印出第二个元素
-v 自定义变量

1.3 sed

参数 含义
-i 直接修改文件内容，而不是打印出来

2.git相关

参数 含义
checkout 切换分支
checkout -b 基于本分支创建新分支并切换
cherry-pick 将别的分支的修改pick到自己分支
push orign –delete xxx 删除分支(远程)
branch -D xxx 删除分支(本地)
rebase 同步master改动到自己分支
